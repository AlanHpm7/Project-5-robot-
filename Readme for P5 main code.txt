Key Features:

DC Motor Control:

'A': Move forward.
'B': Stop.
'C': Move backward.


Stepper Motor Control:

'1': Full step clockwise.
'2': Full step counterclockwise.
'3': Half step clockwise.
'4': Half step counterclockwise.
'0': Turn off stepper motor.


RC Servo Control:

'7': Decrease the angle (tilt down).
'9': Increase the angle (tilt up).
'8': Center the servo.


LED Control:

'*': Toggle LED (optional)
